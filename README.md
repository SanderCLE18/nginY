nginY is a lightweight, HTTP/HTTPS reverse proxy server built to explore the fundamentals of network programming and concurrent connection handling.
While it lacks the complex load-balancing algorithms of its namesake (nginx), it provides a robust core for routing traffic.

Pre-requisites.
- GCC/Clang
- OpenSSL development headers
- CMake

Installation: 
- Run the installation.sh script
- Place websites in the /www/ folder
- Update the config file to your liking.
